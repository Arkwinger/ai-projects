SMS Spam Classifier (Naive Bayes + NLP)

This project uses Natural Language Processing (NLP) and a Multinomial Naive Bayes model to classify SMS messages as spam or ham.

 Features:

Text preprocessing (lowercasing, regex cleaning)

Tokenization

Stopword removal

Stemming

CountVectorizer with unigrams & bigrams

Naive Bayes spam classifier

Hyperparameter tuning with GridSearchCV

Model saved using joblib

 Technologies

Python

Jupyter Notebook

Scikit-learn

NLTK

Pandas

Joblib

 Files

spam_detector.ipynb – full notebook with preprocessing, training, evaluation

spam_detection_model.joblib – the final trained model
